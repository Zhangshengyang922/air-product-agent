## 快速开始

### 方法一：自动同步服务（推荐）

**步骤1：启动同步服务**
```bash
双击运行: 启动企业微信文档同步.bat
```

**步骤2：导出文档**
1. 点击上面的文档链接
2. 打开后点击右上角"文件"按钮
3. 选择"导出" → "CSV"格式
4. 将导出的文件保存到项目目录，重命名为：
   ```
   exported_from_wechat.csv
   ```
5. 保存到：`C:\Users\Administrator\OneDrive\桌面\air_prd_agent\`

**步骤3：自动生效**
- 同步服务会自动检测新导出的文件
- 检测到后自动更新到 `products.csv`
- 前端页面会自动刷新显示最新数据

---

### 方法二：手动复制文件

**步骤1：导出文档**
1. 打开企业微信文档链接
2. 文件 → 导出 → CSV格式

**步骤2：手动复制**
```bash
# 1. 删除旧的 products.csv
del assets\products.csv

# 2. 复制导出的文件
copy exported_from_wechat.csv assets\products.csv

# 3. 重启服务器
# 在服务器窗口按 Ctrl+C 停止
# 重新运行: python src/main.py -m http -p 8000
```

## 工作原理

```
企业微信在线文档
    ↓ 手动导出
exported_from_wechat.csv
    ↓ 自动同步服务检测
assets/products.csv
    ↓ 文件监控
后端重新加载数据
    ↓ WebSocket推送
前端自动刷新
```

### 同步服务特点

- ✅ **自动检测**：每分钟检查一次导出文件
- ✅ **增量更新**：只在文件更新时同步
- ✅ **自动备份**：同步前备份旧文件
- ✅ **实时刷新**：同步后前端立即更新
- ✅ **持续运行**：保持后台监控

## 日常使用流程

### 场景1：更新产品数据

1. 打开企业微信文档：[点击打开](https://doc.weixin.qq.com/sheet/e3_AagAaAYPADkCNj1WqY0AVR2K3kGJf?scode=AAgAKQfkAAs3XlSStd)
2. 修改需要更新的数据（如价格、政策等）
3. 点击"文件" → "导出" → 选择CSV格式
4. 保存到项目目录，命名为 `exported_from_wechat.csv`
5. ✅ 完成！前端会在1分钟内自动刷新

### 场景2：新增产品数据

1. 在企业微信文档中添加新产品行
2. 填写完整的产品信息（航司、名称、舱位、价格等）
3. 导出CSV并保存为 `exported_from_wechat.csv`
4. ✅ 完成！新产品会自动显示在前端

### 场景3：删除产品数据

1. 在企业微信文档中删除相应行
2. 导出CSV并保存
3. ✅ 完成！前端会自动移除该产品

- **products.csv.bak**: 同步前的备份文件（自动生成）

## 故障排查

### 问题1：导出文件后，前端没有更新

**检查步骤：**
1. 确认文件已保存到项目目录
2. 确认文件名为 `exported_from_wechat.csv`
3. 查看同步服务日志，是否有"检测到新的导出文件"
4. 检查 `assets/products.csv` 是否已更新（查看修改时间）

**解决方法：**
- 确保同步服务正在运行
- 检查文件权限
- 手动重启同步服务

### 问题2：同步服务提示"文件不存在"

**原因：**
- `exported_from_wechat.csv` 文件不在项目目录
- 文件名拼写错误

**解决：**
- 确认导出文件保存位置
- 确认文件名完全一致（包括后缀）

### 问题3：导出的文件格式不对

**症状：**
- 打开CSV文件乱码
- 数据无法正确读取

**解决：**
- 导出时选择UTF-8编码
- 确认CSV的列名与系统一致
- 检查是否有特殊字符

### 问题4：同步服务检测不到文件更新

**检查：**
1. 文件是否完整保存（检查文件大小）
2. 文件是否有读写权限
3. 是否有多个同名文件

**解决：**
- 确保Excel/其他程序已关闭文件
- 重新导出并保存
- 重启同步服务

## 进阶使用

### 1. 调整同步间隔

编辑 `sync_wechat_doc.py`：
```python
SYNC_INTERVAL = 30  # 改为30秒检查一次
```

### 2. 多个文档同步

如果需要同步多个文档：
```python
# 添加多个导出文件路径
EXPORTED_FILES = [
    PROJECT_ROOT / "exported_from_wechat.csv",
    PROJECT_ROOT / "exported_products_2.csv",
]

# 在主循环中检查所有文件
for export_file in EXPORTED_FILES:
    if check_export_file(export_file):
        sync_exported_file(export_file)
```

### 3. 自动导出（需要企业微信API）

如果有企业微信文档API权限，可以实现：
```python
import requests

def fetch_wechat_doc():
    """通过API获取文档数据"""
    url = "https://doc.weixin.qq.com/api/export"
    headers = {"Authorization": "Bearer YOUR_TOKEN"}
    response = requests.get(url, headers=headers)
    # 保存为CSV
    with open("exported_from_wechat.csv", "wb") as f:
        f.write(response.content)
```
