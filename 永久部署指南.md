## 🚀 方案1：Railway（最推荐，国内访问快）

### 特点
- 🇨🇳 国内访问速度快
- 💰 免费版支持1个服务
- 🔄 自动HTTPS
- 📱 支持移动端
- 🔧 Docker自动部署

### 部署步骤

#### 第1步：注册账号
1. 访问：https://railway.app
2. 使用GitHub账号登录（推荐）
3. 账号免费

#### 第2步：部署项目
1. 点击 "New Project" → "Deploy from GitHub repo"
2. 如果没有仓库，先创建：
   ```
   # 在Railway页面点击 "Create New Project"
   # 选择 "Deploy from CLI" 或上传代码
   ```

#### 第3步：快速部署（使用Railway CLI）

在本地执行：

```bash
# 安装Railway CLI
npm install -g @railway/cli

# 登录Railway
railway login

# 初始化项目（在项目目录下）
railway init

# 上传并部署
railway up
```

#### 第4步：配置环境
1. 在Railway控制台，找到项目
2. 进入 "Variables" 标签页
3. 设置端口映射：
   - 如果Railway自动分配了随机端口，需要在配置中指定8000

#### 第5步：获取公网地址
1. Railway会自动生成公网URL
2. 格式类似：`https://your-app.railway.app`
3. 这是永久地址，关闭本地电脑也能访问

## 🌐 方案2：Render（免费，稳定）

### 特点
- 💰 完全免费
- 🔗 自动HTTPS证书
- 📱 支持移动端
- ⚡ 自动扩展

### 部署步骤

#### 第1步：注册账号
1. 访问：https://render.com
2. 使用GitHub/GitLab/Google账号登录

#### 第2步：创建Web Service
1. 点击 "New" → "Web Service"
2. 连接GitHub仓库（或上传代码）

#### 第3步：配置部署
在 `render.yaml` 中配置：

```yaml
services:
  - type: web
    name: air-agent
    env: python
    buildCommand: pip install -r requirements.txt
    startCommand: uvicorn main:app --host 0.0.0.0 --port $PORT
    envVars:
      - key: PORT
        value: 8000
```

#### 第4步：获取公网地址
1. Render部署后会生成URL
2. 格式：`https://your-app.onrender.com`
3. 这是永久地址

## 🐳 方案3：腾讯云/CVM（国内访问最快）

### 特点
- 🇨🇳 国内服务器，速度最快
- 🔒 完全控制
- 💎 可选高配置
- 📞 官方中文支持

### 部署步骤

#### 第1步：购买服务器
1. 访问：https://cloud.tencent.com
2. 购买云服务器（CVM）
3. 推荐配置：
   - CPU: 2核
   - 内存: 4GB
   - 带宽: 5Mbps
   - 系统: Ubuntu 22.04
   - 费用: 约￥100-200/月

#### 第2步：远程连接服务器
```bash
# 使用SSH连接（Windows用PuTTY）
ssh root@服务器IP

# 输入密码（购买时设置）
```

#### 第3步：安装环境
```bash
# 更新系统
apt update && apt upgrade -y

# 安装Python
apt install python3.11 python3-pip -y

# 安装依赖
pip3 install fastapi uvicorn pandas

# 上传代码（使用SCP或SFTP）
# 上传整个 air_prd_agent 文件夹到服务器
```

#### 第4步：启动服务
```bash
# 进入项目目录
cd air_prd_agent

# 后台运行服务
nohup python3 cloud_main_v2.py > server.log 2>&1 &

# 查看日志
tail -f server.log
```

#### 第5步：配置防火墙
```bash
# 开放8000端口
ufw allow 8000/tcp

# 验证防火墙
ufw status
```

#### 第6步：访问
```
http://服务器公网IP:8000
```

## 📱 方案4：使用Docker部署（所有平台通用）

### 准备工作
项目已包含 `Dockerfile` 和 `requirements.txt`

### 部署到任意支持Docker的平台

#### 部署到Railway（推荐）
```bash
# 项目根目录执行
railway up
```

#### 部署到Render
1. 在项目根目录创建 `Dockerfile`（已创建）
2. 连接GitHub仓库
3. Render自动识别Dockerfile并部署

#### 部署到腾讯云
```bash
# 构建Docker镜像
docker build -t air-agent .

# 运行容器
docker run -d -p 8000:8000 --name air-agent air-agent

# 查看状态
docker ps
```
