# 航司显示问题解决方案

## 问题描述
左侧航司栏的二字码或名称依旧没有显示。

## 问题分析

### 已确认正确的部分
✅ **AIRLINE_NAMES 映射**: 已更新为36个航司，全部正确
✅ **关键航司**: A6(红土航空), GS(天津航空), DR(桂林航空), LT(龙江航空), GY(多彩贵州)
✅ **JavaScript逻辑**: `updateAirlineDropdown` 和 `renderAirlines` 函数完整
✅ **API调用**: `/api/airlines` 端点存在
✅ **HTML结构**: 航司下拉框元素存在

### 问题根源
❌ **浏览器缓存**: 旧版本的JavaScript被缓存

## 解决方案

### 方法1: 强制刷新（最简单）

**Chrome/Edge**:
1. 按 **Ctrl + Shift + R**
2. 或按 **Ctrl + F5**

**Firefox**:
1. 按 **Ctrl + Shift + R**
2. 或按 **Ctrl + F5**

**Safari (Mac)**:
1. 按 **Cmd + Shift + R**

---

### 方法2: 清除浏览器缓存

**Chrome/Edge**:
1. 按 **Ctrl + Shift + Delete**
2. 选择"缓存的图片和文件"
3. 时间范围选择"全部"
4. 点击"清除数据"
5. 刷新页面

**Firefox**:
1. 按 **Ctrl + Shift + Delete**
2. 选择"缓存"
3. 时间范围选择"全部"
4. 点击"立即清除"

---

### 方法3: 无痕模式测试

**Chrome/Edge**:
1. 按 **Ctrl + Shift + N**
2. 访问 http://localhost:8000
3. 登录（YNTB / yntb123）
4. 查看航司下拉框

**Firefox**:
1. 按 **Ctrl + Shift + P**
2. 访问 http://localhost:8000
3. 登录
4. 查看航司下拉框

**在无痕模式下应该正常显示，如果无痕模式正常，说明是缓存问题**

---

### 方法4: 开发者工具禁用缓存

**Chrome/Edge/Firefox**:
1. 按 **F12** 打开开发者工具
2. 在开发者工具的"Network"或"网络"标签
3. 勾选 "Disable cache" 或 "禁用缓存"
4. 保持开发者工具打开
5. 按 **Ctrl + R** 刷新页面

---

### 方法5: 手动刷新JavaScript

**开发者工具**:
1. 按 **F12** 打开开发者工具
2. 在"Console"（控制台）标签
3. 输入以下命令并回车：
   ```javascript
   location.reload(true);
   ```

### 航司下拉框更新逻辑

**updateAirlineDropdown函数** (第1116-1137行):
```javascript
function updateAirlineDropdown(searchText) {
    const list = document.getElementById('airlineDropdownList');
    
    if (!searchText) {
        // 无搜索时显示全部
        list.innerHTML = `
            <li class="el-select-dropdown__item" data-value="">全部</li>
            ${allAirlines.map(airline => {
                const name = AIRLINE_NAMES[airline.code] || airline.name || airline.code;
                return `<li>...>${airline.code} - ${name}</li>`;
            }).join('')}
        `;
    } else {
        // 搜索过滤
        const filtered = allAirlines.filter(airline => {
            const code = airline.code.toLowerCase();
            const name = (AIRLINE_NAMES[airline.code] || airline.name || '').toLowerCase();
            const search = searchText.toLowerCase();
            return code.includes(search) || name.includes(search);
        });
        // ...渲染过滤结果
    }
}
```
