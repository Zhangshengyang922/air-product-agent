## 🔍 故障排查

### 局域网无法访问

**检查清单**：

1. [ ] 防火墙是否放行8000端口
   ```bash
   netsh advfirewall firewall show rule name="产品管理系统8000"
   ```

2. [ ] IP地址是否正确
   ```bash
   python get_ip.py
   ```

3. [ ] 服务是否正常运行
   ```bash
   netstat -ano | findstr :8000
   ```

4. [ ] 是否在同一局域网
   ```bash
   ping 192.168.101.2
   ```

---

### IP地址变化

**解决方法**：

1. 设置固定IP（推荐）
2. 每次启动前查看IP：`python get_ip.py`
3. 使用DNS服务
