## 公网访问方案

### 方案1：使用ngrok（推荐，最简单）

**步骤：**

1. **下载ngrok**
   - 访问: https://ngrok.com/download
   - 下载Windows版本（如：ngrok-v3-stable-windows-amd64.zip）

2. **解压并运行**
   ```powershell
   # 解压到某个文件夹
   # 打开PowerShell，进入ngrok目录
   ./ngrok http 8000
   ```

3. **获取公网地址**
   - ngrok会显示一个Forwarding URL，例如:
     ```
     Forwarding  https://a1b2c3d4.ngrok.io -> http://localhost:8000
     ```
   - 复制 `https://a1b2c3d4.ngrok.io` 这个地址即可公网访问

4. **分享给他人**
   - 将ngrok提供的URL发送给其他用户
   - 他们通过浏览器打开即可访问

**优点：**
- 无需路由器配置
- 免费版支持HTTP隧道
- 跨平台（Windows/Mac/Linux）

**注意：**
- ngrok免费版URL每次重启会变化
- 需要保持ngrok窗口打开

---

### 方案2：使用Cloudflare Tunnel（推荐，稳定）

**步骤：**

1. **下载cloudflared**
   - 访问: https://github.com/cloudflare/cloudflared/releases
   - 下载最新版: cloudflared-windows-amd64.exe

2. **快速启动（无需登录）**
   ```powershell
   # 下载后，在PowerShell中运行
   ./cloudflared tunnel --url http://localhost:8000
   ```

3. **获取公网地址**
   - cloudflared会显示一个trycloudflare.io的URL
   - 复制该URL即可访问

**优点：**
- Cloudflare官方，稳定快速
- 无需注册即可使用临时隧道
- 支持HTTPS

---

### 方案3：路由器端口映射（永久）

**步骤：**

1. **查看本机信息**
   ```
   本机IP: 192.168.x.x
   端口: 8000
   ```

2. **登录路由器**
   - 打开浏览器，访问路由器管理地址（如：192.168.1.1）
   - 输入管理员账号密码登录

3. **配置端口映射**
   找到以下选项之一：
   - "虚拟服务器" / "端口映射" / "端口转发"
   - "NAT设置" / "外部访问"

   添加规则：
   ```
   外部端口: 8000
   内部端口: 8000
   内部IP: 192.168.x.x（本机IP）
   协议: TCP
   状态: 启用
   ```

4. **获取公网IP**
   - 访问: https://www.ip.cn 或 https://ipinfo.io
   - 得到公网IP，如：123.45.67.89

5. **公网访问地址**
   ```
   http://123.45.67.89:8000
   ```

**优点：**
- 永久固定地址
- 无需运行额外软件
- 速度最快

**注意：**
- 需要路由器管理员权限
- 某些运营商（如电信移动）可能封禁80/8080/8000等端口

## 技术支持

如遇问题，请检查：

1. **本地服务是否运行**
   - 打开浏览器访问 http://localhost:8000
   - 应该能正常登录

2. **端口8000是否占用**
   ```powershell
   netstat -ano | findstr :8000
   ```
   应该看到LISTENING状态

3. **防火墙设置**
   - Windows防火墙需要允许8000端口入站连接
   - 或暂时关闭防火墙测试
