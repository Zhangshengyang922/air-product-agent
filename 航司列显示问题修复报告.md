# 产品表格航司列显示问题修复报告

## 问题描述

用户反馈:产品表格左侧的"航空公司"列只显示航司二字码(如 "MU"),没有显示航司中文名称(如 "中国东方航空")。

## 问题原因

在 `renderTableRow` 函数中(第1376行),航司列只显示了 `product.airline` 字段(航司代码),没有显示航司名称。

## 修复方案

### 1. 修改航司列显示逻辑

**修改前:**
```javascript
<td style="text-align: center;">
    <span class="el-tag el-tag--info" style="font-size: 10px; padding: 2px 6px;">${product.airline}</span>
</td>
```

**修改后:**
```javascript
<td style="text-align: center;">
    <div style="font-size: 11px;">
        <strong style="color: #409eff;">${product.airline}</strong>
    </div>
    <div style="font-size: 10px; color: #606266; margin-top: 1px;">
        ${AIRLINE_NAMES[product.airline] || product.airline}
    </div>
</td>
```

### 2. 调整航司列宽度

由于现在航司列需要显示两行内容(航司代码和中文名称),将列宽从 60px 增加到 100px。

**修改前:**
```javascript
<th style="width: 60px;">航空公司</th>
```

**修改后:**
```javascript
<th style="width: 100px;">航空公司</th>
```

## 显示效果

修复后,航司列会显示:
- 第一行:航司二字码(蓝色加粗) - 例如: **MU**
- 第二行:航司中文名称(灰色) - 例如: 中国东方航空

## 修改的文件

- `static/index_elui.html`
  - 第1313行:航司列标题宽度从60px改为100px
  - 第1375-1377行:航司列显示逻辑,从只显示代码改为显示代码和中文名称

## 验证方法

刷新页面后,产品表格的"航空公司"列应该显示:
- **MU**
  中国东方航空
- **CZ**
  中国南方航空
- **CA**
  中国国际航空
- ...

所有航司都会显示其二字码和中文名称。如果某个航司代码不在 `AIRLINE_NAMES` 映射中,则只显示代码。

## 注意事项

1. 需要强制刷新浏览器以加载更新后的代码 (Ctrl + Shift + R)
2. 已添加缓存控制meta标签,未来更新将自动生效
3. 列宽调整为100px以容纳两行内容
