# 航空公司产品智能体系统

## 简介

欢迎使用航空公司产品智能体系统！这是一个基于LangChain框架的智能助手，专门用于管理和查询航空公司产品信息。

---

## 主要功能

✅ **产品查询** - 根据航空公司、航线、舱位等条件查询产品
✅ **智能搜索** - 支持关键词搜索，快速找到目标产品
✅ **统计分析** - 提供产品统计信息和数据概览
✅ **文档查询** - 内置帮助文档，随时获取使用指南
✅ **数据导出** - 一键导出产品数据到Excel

---

## 技术栈

- **框架**: LangChain + LangGraph
- **语言**: Python 3.8+
- **数据处理**: Pandas
- **文档**: Markdown
- **导出**: Excel (openpyxl)

---

## 项目结构

```
/workspace/projects/
├── config/                  # 配置文件
├── src/
│   ├── agents/             # 智能体代码
│   ├── storage/            # 存储管理
│   └── tools/              # 工具函数
└── assets/
    ├── cleaned_data/       # 数据文件
    ├── docs/               # 文档
    └── agent_data_with_docs/  # 导出数据
```
