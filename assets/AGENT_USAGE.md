# 智能体使用指南

## 快速开始

### 1. 查询产品

你可以通过以下方式查询产品：

**按航空公司查询**:
```
查询东方航空(MU)的所有产品
```

**关键词搜索**:
```
搜索所有经济舱产品
```

**按航线查询**:
```
查找北京到上海的产品
```

---

### 2. 获取统计信息

```
给我看一下统计数据
```

智能体会返回：
- 产品总数
- 航空公司数量
- 各航空公司的产品分布

---

### 3. 导出数据

```
帮我导出所有产品数据
```

数据会导出到 `assets/agent_data_with_docs/所有产品数据_export.xlsx`

---

### 4. 获取帮助

```
我该如何使用查询功能？
```

智能体会返回相关的帮助文档。

---

## 常用场景

### 场景1：比较不同航空公司的产品

```
帮我想看看东航和南航的经济舱产品
```

智能体会：
1. 查询东航产品
2. 查询南航产品
3. 筛选经济舱产品
4. 对比展示结果

---

### 场景2：查找特定航线

```
有没有北京到广州的航班产品？
```

智能体会搜索所有相关的航线产品。

---

### 场景3：了解产品政策

```
东航里程优享产品有什么政策？
```

智能体会搜索产品备注中的政策信息。

---

## 对话技巧

1. **使用自然语言**: 不需要记住具体的API命令，用自然语言描述你的需求即可
2. **逐步细化**: 先问大的问题，再逐步细化
3. **指定关键信息**: 如果知道具体航空公司或航线，直接告诉智能体
4. **请求帮助**: 遇到问题时随时询问"如何..."

---

## 示例对话

**用户**: 有哪些航空公司？

**智能体**: [调用 get_all_airlines 工具]
当前系统中有以下航空公司：MU, CZ, HU, CA, FM, ZH, 3U, MF ...

---

**用户**: 东航有多少个产品？

**智能体**: [调用 get_products_by_airline 工具]
东方航空(MU)共有 25 个产品...

---

**用户**: 帮我找一下所有带"里程"的产品

**智能体**: [调用 search_airline_products 工具]
找到 10 个包含"里程"的产品：
1. MU - 1、里程优享 - 所有航线
2. CZ - 2、里程兑换产品 - 国内航线
...

---

## 注意事项

1. 搜索是**不区分大小写**的
2. 支持**中英文混合**搜索
3. 搜索结果会限制返回数量（默认10条）
4. 导出的Excel文件保存在 `assets/agent_data_with_docs/` 目录下
