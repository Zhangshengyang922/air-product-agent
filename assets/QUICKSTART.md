# 快速开始指南

## 5分钟上手

### 第一步：了解智能体

这是一个航空公司产品智能助手，可以帮助你：
- 查询航班产品信息
- 搜索特定航线或舱位
- 了解产品政策
- 导出产品数据

---

### 第二步：基本查询

**查询所有航空公司**:
```
有哪些航空公司？
```

**查询某个航空公司的产品**:
```
帮我看看东航的产品
```

**搜索特定产品**:
```
搜索经济舱产品
```

---

### 第三步：获取帮助

如果不知道如何使用，随时可以问：
```
我该怎么用？
```
或
```
查看帮助文档
```

---

## 常见问题

### Q: 如何查找某个航线的航班？

A: 直接告诉智能体你的出发地和目的地：
```
有没有北京到上海的航班？
```

### Q: 如何了解产品的价格信息？

A: 智能体会在查询结果中自动显示价格信息（上浮价格）。

### Q: 如何导出数据？

A: 简单地说：
```
导出所有产品数据
```

### Q: 支持哪些航空公司？

A: 询问：
```
有哪些航空公司？
```
