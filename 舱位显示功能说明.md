# 舱位显示功能说明

## 功能概述

在前端产品列表表格中新增了舱位显示和筛选功能,用户可以:
- 在表格主列中查看舱位信息
- 根据舱位等级进行筛选
- 通过颜色区分不同舱位等级

### 2. 舱位筛选功能

#### 筛选器位置
在筛选栏中添加了"舱位等级"筛选器,位于"票证类型"和"产品类型"之间。

#### 筛选逻辑
```javascript
// 舱位等级定义
const CABIN_CLASSES = {
    'ECONOMY': ['Y', 'B', 'M', 'H', 'K', 'L', 'Q', 'T', 'X', 'U', 'E', 'W', 'S', 'N', 'R'],
    'BUSINESS': ['C', 'D', 'I', 'Z', 'J'],
    'FIRST': ['F', 'A', 'P']
};

// 判断舱位等级
function getCabinClassLevel(bookingClass) {
    if (!bookingClass) return null;
    const bc = bookingClass.toUpperCase();
    
    for (const [level, classes] of Object.entries(CABIN_CLASSES)) {
        if (classes.includes(bc)) {
            return level;
        }
    }
    return null;
}
```

#### 3.2 表格行渲染
```javascript
// 舱位显示样式
let cabinClassDisplay = bookingClass;
let cabinClassStyle = '';

// 根据舱位类型设置不同颜色
if (bookingClass) {
    const upperBookingClass = bookingClass.toUpperCase();
    if (['Y', 'B', 'M', 'H', 'K', 'L', 'Q', 'T', 'X', 'U', 'E', 'W', 'S', 'N', 'R'].includes(upperBookingClass)) {
        cabinClassStyle = 'color: #67c23a; font-weight: 500;'; // 经济舱 - 绿色
    } else if (['C', 'D', 'I', 'Z', 'J'].includes(upperBookingClass)) {
        cabinClassStyle = 'color: #e6a23c; font-weight: 500;'; // 商务舱 - 橙色
    } else if (['F', 'A', 'P'].includes(upperBookingClass)) {
        cabinClassStyle = 'color: #f56c6c; font-weight: 500;'; // 头等舱 - 红色
    } else {
        cabinClassStyle = 'color: #909399; font-weight: 500;'; // 其他 - 灰色
    }
}

// 在表格中显示
<td style="text-align: left;">
    <div style="${cabinClassStyle}">${cabinClassDisplay || '-'}</div>
</td>
```

#### 3.3 筛选逻辑
```javascript
function filterProducts(products) {
    let filtered = [...products];
    
    // ... 其他筛选逻辑 ...
    
    if (selectedCabinClass) {
        filtered = filtered.filter(p => {
            if (!p.booking_class) return false;
            const cabinLevel = getCabinClassLevel(p.booking_class);
            return cabinLevel === selectedCabinClass;
        });
    }
    
    // ... 其他筛选逻辑 ...
    
    return filtered;
}
```

## 使用说明

### 查看舱位
1. 登录系统后访问主页
2. 产品列表会显示所有产品的舱位信息
3. 舱位以不同颜色显示:
   - 绿色: 经济舱
   - 橙色: 商务舱
   - 红色: 头等舱
   - 灰色: 其他

### 按舱位筛选
1. 在筛选栏中找到"舱位等级"下拉框
2. 选择要筛选的舱位等级:
   - 全部: 显示所有舱位
   - 经济舱: 只显示经济舱产品
   - 商务舱: 只显示商务舱产品
   - 头等舱: 只显示头等舱产品
3. 产品列表会实时更新显示筛选结果

### 组合筛选
舱位筛选可以与其他筛选器组合使用:
- 航空公司 + 舱位等级
- 票证类型 + 舱位等级
- 产品类型 + 舱位等级
- Office号 + 舱位等级
- 所有条件的组合

## 舱位代码参考

### 经济舱 (Economy Class)
最常见的经济舱代码:
- **Y** - 全价经济舱
- **B, M** - 折扣经济舱
- **H, K, L** - 淡季经济舱
- **Q, T, X** - 特价经济舱
- **U, E, W, S, N, R** - 其他经济舱代码

### 商务舱 (Business Class)
- **C** - 全价商务舱
- **D, I** - 折扣商务舱
- **Z, J** - 淡季商务舱

### 头等舱 (First Class)
- **F** - 全价头等舱
- **A, P** - 折扣头等舱

## 数据说明

### 数据字段
舱位信息来自产品的 `booking_class` 字段:
```javascript
product.booking_class
```

### 数据示例
```json
{
    "airline": "CA",
    "product_name": "国内经济舱产品",
    "booking_class": "Y",
    "policy_code": "CA001",
    ...
}
```

### 空值处理
如果产品的舱位信息为空或不存在,显示为 `-`。
