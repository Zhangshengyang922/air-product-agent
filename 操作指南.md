## 🔄 未来更新产品文件的方式

**适用场景**：从企业微信或其他Excel源更新数据

2. **运行导入脚本**

3. **查看导入结果**

   - 访问 http://localhost:8000

**优点**：
- ✅ 支持多工作表
- ✅ 自动清理数据
- ✅ 编码自动转换
- ✅ 自动备份旧数据

## 🖥️ 开机启动与局域网访问

### 开机自动启动

#### 方法1：使用Windows任务计划（推荐）

**操作步骤**：

1. **打开任务计划程序**
   - Win+R，输入 `taskschd.msc`
   - 回车打开

2. **创建基本任务**
   - 点击右侧"创建基本任务"
   - 名称：`产品管理系统`
   - 点击"下一步"

3. **设置触发器**
   - 选择"当计算机启动时"
   - 点击"下一步"

4. **设置操作**
   - 选择"启动程序"
   - 程序或脚本：
     ```
     C:\Users\Administrator\AppData\Local\Programs\Python\Python313\python.exe
     ```
   - 添加参数：
     ```
     "C:\Users\Administrator\OneDrive\桌面\air_prd_agent\src\main.py" -m http -p 8000
     ```
   - 起始于：
     ```
     C:\Users\Administrator\OneDrive\桌面\air_prd_agent
     ```
   - 点击"下一步"

5. **完成**
   - 点击"完成"
   - 勾选"当单击完成时，打开此任务属性的对话框"
   - 点击"完成"

6. **配置高级设置**
   - 在"常规"选项卡
   - 勾选"使用最高权限运行"
   - 配置为：Windows 10
   - 点击"确定"

---

#### 方法2：创建启动快捷方式

**操作步骤**：

1. **创建批处理文件**
   ```batch
   @echo off
   chcp 65001
   cd "C:\Users\Administrator\OneDrive\桌面\air_prd_agent"
   python src\main.py -m http -p 8000
   pause
   ```
   保存为 `start_server.bat`

2. **复制到启动文件夹**
   - Win+R，输入 `shell:startup`
   - 将 `start_server.bat` 复制到打开的文件夹中

3. **重启电脑**
   - 系统启动时会自动运行服务

### 局域网访问配置

#### 步骤1：获取本机IP地址

```bash
# 查看IP地址
ipconfig
```

找到"无线局域网适配器"或"以太网适配器"下的IPv4地址，例如：
```
IPv4 地址 . . . . . . . . . . . : 192.168.1.100
```

#### 步骤2：修改服务启动命令

```bash
# 使用 0.0.0.0 监听所有网络接口
python src\main.py -m http -p 8000 --host 0.0.0.0
```

**注意**：默认的 `uvicorn` 已经使用 `0.0.0.0`，所以无需额外配置。

#### 步骤3：配置防火墙

**Windows防火墙配置**：

1. **打开防火墙设置**
   - 控制面板 → 系统和安全 → Windows Defender 防火墙
   - 点击"高级设置"

2. **添加入站规则**
   - 点击"入站规则"
   - 点击右侧"新建规则"

3. **配置规则**
   - 规则类型：端口
   - 协议：TCP
   - 特定本地端口：8000
   - 操作：允许连接
   - 配置文件：域/专用/公用（全部勾选）
   - 名称：产品管理系统8000
   - 点击"完成"

---

#### 步骤4：局域网访问

在同一局域网内的其他电脑上，打开浏览器访问：

```
http://192.168.1.100:8000
```

替换 `192.168.1.100` 为您实际的IP地址。

---

#### 步骤5：固定IP地址（可选）

为了方便局域网访问，建议设置固定IP：

1. **打开网络适配器设置**
   - 控制面板 → 网络和Internet → 网络和共享中心
   - 点击"更改适配器设置"

2. **配置网卡**
   - 右键点击网卡 → 属性
   - 双击"Internet 协议版本 4 (TCP/IPv4)"

3. **设置固定IP**
   - 选择"使用下面的IP地址"
   - IP地址：`192.168.1.100`（建议选择未被占用的IP）
   - 子网掩码：`255.255.255.0`
   - 默认网关：路由器IP（如 `192.168.1.1`）
   - DNS服务器：`114.114.114.114` 或 `8.8.8.8`
   - 点击"确定"

## 🔧 故障排查

### 问题1：服务无法启动

**症状**：运行启动命令后报错

**解决方案**：

1. **检查端口占用**
   ```bash
   netstat -ano | findstr :8000
   ```

2. **结束占用进程**
   ```bash
   taskkill /F /PID <进程ID>
   ```

3. **检查Python环境**
   ```bash
   python --version
   pip list
   ```

4. **查看错误日志**
   ```bash
   type server.log
   ```

---

### 问题2：局域网无法访问

**症状**：本机可以访问，其他电脑无法访问

**解决方案**：

1. **检查防火墙**
   - 确认8000端口已放行
   - 或临时关闭防火墙测试

2. **检查IP地址**
   ```bash
   ipconfig
   ```
   确认IP地址正确

3. **检查网络连接**
   ```bash
   ping 192.168.1.100
   ```

4. **检查服务监听地址**
   - 确认服务使用 `0.0.0.0` 监听

---

### 问题3：数据更新不生效

**症状**：更新CSV后Web界面未变化

**解决方案**：

1. **重启服务**
   ```bash
   # 找到进程并结束
   taskkill /F /IM python.exe

   # 重新启动
   python src\main.py -m http -p 8000
   ```

2. **检查文件编码**
   - 确保使用UTF-8编码
   - 推荐使用UTF-8 with BOM

3. **检查文件格式**
   - 确认CSV格式正确
   - 检查字段名称

4. **清除浏览器缓存**
   - Ctrl+F5 强制刷新

---

### 问题4：导入失败

**症状**：运行导入脚本报错

**解决方案**：

1. **检查文件名**
   - 确保文件名为 `exported_from_wechat.xlsx`
   - 放在项目根目录

2. **检查文件格式**
   - 确保是有效的Excel文件
   - 可以先用Excel打开验证

3. **检查依赖包**
   ```bash
   pip install openpyxl
   ```

4. **查看详细错误**
   - 检查控制台输出
   - 查看traceback信息
