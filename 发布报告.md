## API接口文档

### 主要接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/products | GET | 获取所有产品 |
| /api/products/airline/{airline} | GET | 按航空公司筛选 |
| /api/products/search | GET | 搜索产品 |
| /api/statistics | GET | 获取统计数据 |
| /api/upload/products | POST | 上传产品文件 |

### 接口示例

#### 获取所有产品
```bash
GET http://localhost:8000/api/products
```

#### 按航空公司筛选
```bash
GET http://localhost:8000/api/products/airline/MU
```

#### 搜索产品
```bash
GET http://localhost:8000/api/products/search?keyword=里程优享
```
