# 产品数据更新使用说明

## 📌 快速开始

### 1. 更新产品数据

将CSV文件上传到项目根目录，命名为 `exported_from_wechat.csv`

然后运行更新脚本：

```bash
python update_products.py
```

或者直接双击运行 `start.bat`，选择选项 1

### 2. 启动Web服务（端口8000）

```bash
python src/main.py -m http -p 8000
```

或者双击运行 `start.bat`，选择选项 2

### 3. 访问Web界面

- 打开浏览器访问：`http://localhost:8000`
- 登录账号：`YNTB` / `yntb123`
- 查看产品数据

---

## 📊 CSV文件格式要求

您的CSV文件应该包含以下字段：

| 字段名 | 说明 | 示例 |
|--------|------|------|
| 产品名称 | 产品完整名称 | CZ优享飞2024版 |
| 航线 | 适用航线 | KMG319/KMG279 |
| 订座舱位 | 舱位代码 | X/V/Q |
| 上浮价格 | 价格上浮金额 | +50 |
| 政策返点 | 返点比例 | 2% |
| 产品代码 | 产品标识 | CZ-YX-001 |
| 票证类型 | GP/BSP/B2B等 | GP |
| 出票OFFICE | 出票Office号 | CANO922 |
| 备注 | 其他说明 | 适用于团队客户 |
| 产品有限期 | 有效日期范围 | 2024-01-01 至 2024-12-31 |

## 🔧 手动操作步骤

### 方法1：使用更新脚本（推荐）

1. 将CSV文件上传到项目根目录
2. 重命名为 `exported_from_wechat.csv`
3. 运行更新脚本：
   ```bash
   python update_products.py
   ```
4. 重启服务：
   ```bash
   python src/main.py -m http -p 8000
   ```

### 方法2：直接复制文件

1. 将CSV文件直接复制替换 `assets/products.csv`
2. 重启服务：
   ```bash
   python src/main.py -m http -p 8000
   ```

### 方法3：通过Web界面上传

1. 启动服务：`python src/main.py -m http -p 8000`
2. 访问：`http://localhost:8000`
3. 登录后点击"上传产品"按钮
4. 选择CSV文件上传

## ❓ 常见问题

### 1. 文件编码问题

如果遇到编码错误，请确保CSV文件使用以下编码之一：
- UTF-8 (推荐)
- UTF-8 with BOM (utf-8-sig)
- GBK
- GB2312

### 2. 端口冲突

如果8000端口被占用，可以修改端口号：

```bash
python src/main.py -m http -p 其他端口号
```

### 3. 数据未更新

确保：
1. CSV文件命名为 `exported_from_wechat.csv`
2. 文件位于项目根目录
3. 运行了更新脚本
4. 重启了Web服务
